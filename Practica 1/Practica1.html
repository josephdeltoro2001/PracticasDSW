¿QUÉ ES HTML?

HTML son siglas de (Hypertext Markup Lenguaje), que puede traducirse como lenguaje de marcas o marcado de hipertexto

El lenguaje HTML se emplea para crar las páginas web. Es muy fácil ver el código HTML de una página web, la opción exacta cambia de un navegador a otro y también puede cambiar de una versión a otra de un mismo navegador,
pero suelen tener un nombre similar.

El lenguaje HTML sirve para describir la estructura básica de una página y organizar la forma en que se mostrará su contenido, además de que HTML permite incluir enlaces hacia otras páginas o documentos.

HTML se compone de etiquetas que se escriben entre los simbolos menor que y mayor que.

Los inicios del lenguaje HTML se remontan al año 1990, cuando Tim Berners-Lee creó la primera página web.
